# 毕业论文管理系统

**技术栈：** Java 21 / Spring Boot 3.2 / JPA / MySQL / Redis / JWT / Tailwind CSS

独立设计并开发了一套面向高校毕业论文全流程的管理系统，覆盖选题申报、开题报告、中期检查、终稿提交、答辩评分到成绩归档的完整业务闭环。系统支持学生、指导老师、专业负责人、学院领导、评阅老师、答辩小组、管理员七种角色，基于 JWT 实现无状态认证与 RBAC 权限控制，通过 Spring AOP 对 48 个 RESTful API 进行声明式权限校验。

后端采用分层架构（Controller / Service / Repository / Entity / DTO），设计多级审核流程状态机，各角色分阶段审核，材料历史版本可追溯，支持驳回重提。评分体系按指导老师 40%、评阅老师 20%、答辩小组 40% 权重自动汇总最终成绩。前端使用原生 JavaScript + Tailwind CSS 构建单页应用，按角色模块化组织，封装统一 Token 管理与 API 调用。集成 Redis 实现短信验证码找回密码，支持归档材料 ZIP 批量导出。
